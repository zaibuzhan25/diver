### 16:46 — 修复 MySQL TEXT 列不能有 DEFAULT 的报错
- 根因: Destination.java / User.java / TravelPlan.java 使用 `TEXT DEFAULT 'xxx'`，MySQL strict mode 拒绝
- 修复: `columnDefinition = "TEXT DEFAULT '中国'"` → `length = 100`（VARCHAR），Java 字段默认值兜底
- 同时移除 application.properties 中多余的 hibernate.dialect 手动指定（MySQL 8+ 自动检测）
