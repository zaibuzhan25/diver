## 致命语法错误修复 (11:30)

**根因：user.html 所有按钮点不了**

之前修复的都是运行时错误，但漏了最根本的问题——**JS 有语法错误，整页 JS 完全无法执行**：

1. **`function viewDestination(id)` 缺少 `async`** — 函数内用了 `await api(...)` 但没有声明 `async`

   - 所有事件绑定、函数定义全部随之失效
   - 修复：`function viewDestination(id)` → `async function viewDestination(id)`

2. **`onerror` 属性引号损坏** — `loadProfileFavorites` 中 `onerror="this.src='...'"` 的单引号破坏了 JS 字符串
   - 修复：删除 `onerror` 属性，改用 `style` 属性设置尺寸
